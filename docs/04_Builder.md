# Izumo Builder 設計概要

作成日: 2026-06-30 (JST)

---

# 1. 概要

Izumo Builderは、
GUIでISO生成を支援するツールである。

---

# 2. 目的

初心者でもLinuxディストリビューションを
ワンクリックでビルドできるようにする。

---

# 3. 技術構成

- Python3
- GTK4
- live-buildラッパー

---

# 4. 機能（予定）

- ISOビルドボタン
- 設定GUI
- パッケージ選択
- ログ表示
- ビルド進捗表示

---

# 5. ディレクトリ構造

builder/
├── src/
├── ui/
├── resources/
└── __init__.py

---

# 6. 設計方針

- シンプルUI
- 初心者向け
- CLI依存を隠蔽
- 安定性優先
