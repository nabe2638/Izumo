# Izumo Project

## 概要
Izumo Projectは、Raspberry Pi 4で動作する軽量Linux OSを構築するプロジェクトです。

USBで持ち運べるLinux環境を目標とし、初心者でも再現可能な構成を目指しています。

---

## 特徴
- Debian 13 (trixie) ベース
- Raspberry Pi 4対応
- 軽量デスクトップ（Xfce）
- USBブート対応予定
- GitHubで完全再現可能
